Try this

  • Select some text and toggle bold, italic, underline, or strikethrough.
  • Pick a heading from the block-format dropdown to convert this paragraph to a heading.
  • Use the color buttons to change text color or highlight a phrase.
  • Press Ctrl/Cmd + F to open Find & Replace, then replace "sample" with something else.
  • Insert an image by URL with the picture button.
  • Use the alignment buttons to recenter or justify a paragraph.

Lists, quotes, code

  1. Numbered lists work too.
  2. Indent / outdent with the toolbar arrows.
Block quotes are available from the heading dropdown or the quote button.
